Question-1: What are the different ways to select an element in the DOM?

There are 5 ways in which you can select elements in a DOM using selectors.

  • getElementsByTagName()
  • getElementsByClassName()
  • getElementById()
  • querySelector()
  • querySelectorAll()

querySelectorAll() and getElementById() are mostly used in every project.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

textContents, innerText and innerHtml allows to read and update the content of HTML elements. But they have different behaviours in terms of the content they include and how they handle HTML markup.

  • textContents is all text contained by an element and all its children that are for formatting purposes only.
  • innerText returns all text contained by an element and all its child elements.
  • innerHtml returns all text, including html tags, that is contained by an element.

Question-3: What is event delegation in the DOM?

Event Delegation is a pattern based upon the concept of Event Bubbling. It is an event-handling pattern that allows to handle events at a higher level in the DOM tree other than the level where the event was first received.

Question-4: What is event bubbling in the DOM?

Event bubbling is a concept in the DOM where an event that occurs on a nested element propagates to its parent elements. This means that if an event occurs on a child element, it will first trigger any event handlers on that element and then propagate up through its parent, grandparent, etc. until it reaches the root of the document.

Question-5: How do you create, add, and remove elements using JavaScript?

To add, remove, and modify HTML elements using JavaScript, you can use methods like createElement , appendChild , removeChild , and properties like innerHTML and textContent . For example, to add an element, you can create it using document. createElement and then append it to a parent element using appendChild .
